# PCTycoon-for-MacOS-26.5.1-v1
This is a MacOS app for MacOS 26.5.1 Windows 10 and 11 versions will be out soon.
Follow These Steps
1. Open the DMG and drag PCTycoon to Applications
2. Right-click PCTycoon → Open → Open
   (Don’t double-click the first time)
3. After that one-time step, it opens normally

Or after a blocked launch: System Settings → Privacy & Security → Open Anyway
