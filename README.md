# PCTycoon-for-MacOS-26.5.1-v1
This is a MacOS app for MacOS 26.5.1 Windows 10 and 11 is out. They may not work though.
. Copy PCTycoon.dmg to the other Mac (AirDrop, USB, cloud, etc.)
2. Double-click the DMG to open it
3. Drag PCTycoon.app into Applications
4. Eject the DMG
5. Open Applications and find PCTycoon
6. Right-click (or Control-click) PCTycoon → Open
7. Click Open in the dialog (not Cancel)
